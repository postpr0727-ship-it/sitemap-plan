사이트맵 기획 보드

홈페이지 제작을 위한 메뉴/페이지 구조, 상·하위 관계, 병렬/직렬 구성을 설계하세요.

Tailwind + JavaScript

사이트맵 기획 보드에 오신 것을 환영합니다

여러 회사/프로젝트의 홈페이지 구조를 프로젝트별로 나누어 설계하고, 상·하위 메뉴와 URL을 한 번에 정리할 수 있는 도구입니다.

  • 좌측에서 작업 프로젝트를 선택하거나 새로 만듭니다.
  • 페이지별 제목 · URL · 상위 메뉴 · 요약을 입력합니다.
  • 우측에서 사이트맵 구조와 페이지 목록을 함께 확인합니다.
  • 완료되면 상단의 JSON 내보내기 버튼으로 결과를 파일로 저장하세요.

사이트맵 구조 미리보기

- 박스 간 세로 줄기는 상·하위(계층) 관계를 의미합니다.
- 같은 레벨에서 직렬(순차) 그룹은 번호로 이어지며, 병렬 그룹은 가로로 나란히 배치됩니다.
- 각 박스를 클릭하면 해당 페이지가 편집 모드로 전환됩니다.
좌측에서 페이지를 추가하면 사이트맵 구조가 자동으로 그려집니다.
